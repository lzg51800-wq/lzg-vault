# Codex 加工产出区

本目录用于存放 **Codex（ChatGPT 内置）** 读取龙虾简报 + Inbox 后加工产出的内容，供龙虾定时 pull 读回、用于推送。

## 产物两类

1. **每日观点**（保留旧格式）
   - 路径：`Codex观点-YYYY-MM-DD.md`（每日一份，幂等覆盖）
   - 三段：重点观点 / 持仓关注 / 风险提示
2. **原子概念卡片**（新增，自生长核心）
   - 路径：`../02_Knowledge/概念名.md`
   - 一个概念一页，含五维评估 + `[[双向链接]]`
   - 龙虾推送时**优先读此目录**，概念比日期聚合更易复用

## 约定
- **写入方**：Codex（本地 Mac，ChatGPT 内置）
- **读取方**：龙虾（腾讯云，定时 git pull 后读取）
- **冲突防护**：本目录仅 Codex 写，龙虾只读

## 使用流程
1. 在 ChatGPT 里用 Codex 模式，按 `🤖AI/Codex回流指令模板.md` 的自生长版指令运行
2. Codex 读取 `💰金融/自动/` + `Inbox/` → 加工 → 写回本目录 + `💰金融/02_Knowledge/`
3. Obsidian Git 自动备份到 GitHub
4. 龙虾定时 pull → 读取 → 推送微信/飞书
