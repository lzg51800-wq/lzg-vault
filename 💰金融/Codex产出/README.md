# Codex 加工产出区

本目录用于存放 **Codex（ChatGPT 内置）** 读取龙虾简报后加工产出的内容，供龙虾定时 pull 读回、用于推送。

## 约定

- **写入方**：Codex（本地 Mac，ChatGPT 内置）
- **读取方**：龙虾（腾讯云，定时 git pull 后读取）
- **命名**：`Codex观点-YYYY-MM-DD.md`（每日一份，幂等覆盖）
- **格式**：纯 Markdown，含「重点观点 / 持仓关注 / 风险提示」三段，方便龙虾抽取推送
- **冲突防护**：本目录仅 Codex 写，龙虾只读；龙虾自己的简报在 `💰金融/自动/`，互不干扰

## 使用流程

1. 在 ChatGPT 里用 Codex 模式，按 `🤖AI/Codex回流指令模板.md` 的指令运行
2. Codex 读取 `💰金融/自动/` 的龙虾简报 → 加工 → 写回本目录
3. Obsidian Git 自动备份到 GitHub
4. 龙虾定时 pull → 读取本目录 → 推送微信/飞书
