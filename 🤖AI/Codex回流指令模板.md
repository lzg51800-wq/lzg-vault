# Codex 回流指令模板

把下面这段发给 **ChatGPT 里的 Codex**（或 Codex CLI），让它读龙虾简报、加工、写回 vault。

## 指令原文（复制给 Codex）

> 我的 Obsidian 笔记在 `/Users/Apple/Documents/file/LzgFile`。
>
> 请执行以下任务：
> 1. 读取 `💰金融/自动/` 目录下今天及最近的龙虾简报（金融新闻简报 / A股晨报 / A股复盘）。
> 2. 综合提炼出一份「每日观点」，包含三段：
>    - **重点观点**：3-5 条最值得关注的宏观/市场判断
>    - **持仓关注**：结合简报提到的板块/个股，给出关注清单（不荐股，只列观察项）
>    - **风险提示**：当日主要风险点
> 3. 把结果写成 Markdown，保存到 `💰金融/Codex产出/Codex观点-YYYY-MM-DD.md`（用今天日期）。
> 4. 改笔记时保留 `---` frontmatter、双链 `[[ ]]`、标签 `#金融/xxx` 语法不变，只写我要求的内容。
> 5. 完成后告诉我写了什么、文件在哪。

## 使用说明

- 每天龙虾简报同步后（约 17:30 复盘后），跑一次即可
- Codex 直接文件系统读写 vault，零配置
- 写完后 Obsidian Git 会自动备份，龙虾下次 pull 就能读到
- 龙虾读取端见 `🤖AI/环路待办-服务器侧.md`
