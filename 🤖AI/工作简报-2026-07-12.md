## 五、今日踩坑与解法（可复用）

| 坑 | 解法 |
|----|------|
| SSH 私钥末行缺换行 → `invalid format` | `printf '\n' >> key` 补换行 |
| `ssh-copy-id lobster` 报 No identities | 显式 `-i ~/.ssh/lobster_key.pub` |
| coscli v1.0.8 无 `config set`，向导写坏 `~/.cos.yaml` | `cat > ~/.cos.yaml << 'EOF'` 直接覆盖 |
| CAM `cos:ListBucket` action 不存在 | 改用 `cos:GetBucket` / `cos:HeadBucket` |
| 浏览器打开 URL 变下载 | upload 加 `--meta "Content-Type:text/html"` |
| `plugin not installed: tavily` 警告 | CLI 误报，gateway 运行时实际已加载（真实运行已证明） |
| 非交互 SSH 跑 openclaw 报 `node: not found` | 用 `bash -lc` 包裹命令 |
| 沙盒写 `~/.ssh` 被拒 `Operation not permitted` | github_key 改在用户 Mac 终端生成 |
| Obsidian Git "Git is not ready"（缺 author） | `git config user.name/user.email` 命令行写仓库级 |
| Obsidian Git 2.38.6 自动备份字段名 | 叫 `autoSaveInterval`（UI 显示 "Auto commit-and-sync interval"） |
| Obsidian 运行时退出会覆盖 community-plugins.json | 先让用户 Cmd+Q 完全退出，再锁定插件启用配置 |
| Dataview 代码块不显示表格 | 需切到阅读模式（Cmd+E）才渲染 |
