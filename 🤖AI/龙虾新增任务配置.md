# 龙虾新增任务配置（4个）

> 在龙虾 Web 面板 → 自动化 → 新建 → 粘贴自然语言 → 设置时间 → 保存
> 任务 1-2 为定时推送，任务 3-4 为长期指令（发一次微信即可生效）

---

## 任务 1：盘中异动监控（定时推送）

- **类型**：定时任务（cron）
- **时间**：工作日盘中每 30 分钟（9:30-11:30, 13:00-15:00）

### 名称
```
盘中异动监控
```

### 它应该做什么？
```
你是A股盘中监控助手。请执行以下任务：
1. 读取 watchlist.md 获取我关注的自选股列表
2. 使用 westock-data 技能获取每只自选股的实时行情数据：当前价、涨跌幅、成交量、换手率
3. 使用 Infoway REST API 获取个股实时买卖盘五档数据
4. 触发预警的条件（任一满足即报警）：
   - 涨跌幅超过 ±3%
   - 成交量较前一日同时段放大 2 倍以上
   - 换手率超过 5%
   - 出现大单买入/卖出（单笔金额超 100 万）
5. 如果有触发预警的个股，立即微信通知我，格式：
   ⚠️ 异动提醒
   股票名称（代码）
   当前价：XX | 涨跌幅：±X.X%
   触发原因：XXX
   成交量：XX万手 | 换手率：X.X%
   建议：XXX
6. 如果没有异动，不需要发送任何消息（静默运行）
7. 注意：红色=涨、绿色=跌的中国惯例
```

### 时间设置

龙虾面板里用自然语言描述：
```
工作日（周一至周五）交易时间内每 30 分钟执行一次，
具体时间点：9:30, 10:00, 10:30, 11:00, 11:30, 13:00, 13:30, 14:00, 14:30, 15:00
```

如果面板只支持 cron 表达式，用这 3 条：
```
30 9 * * 1-5        （9:30 开盘检查）
0,30 10-11 * * 1-5  （10:00, 10:30, 11:00, 11:30）
0,30 13-14 * * 1-5  （13:00, 13:30, 14:00, 14:30）
0 15 * * 1-5        （15:00 收盘检查）
```

### 推送方式
```
有异动时微信通知，无异动时静默
```

## 任务 2：公告/研报速递（定时推送）

- **类型**：定时任务（cron）
- **时间**：工作日 18:00

### 名称
```
公告研报速递
```

### 它应该做什么？
```
你是A股公告研报分析师。请执行以下任务：
1. 读取 watchlist.md 获取我关注的自选股列表
2. 使用 westock-data 技能采集今日所有上市公司公告，筛选出我自选股相关的公告
3. 使用 Tavily web_search 搜索今日券商研报，关键词为自选股名称+研报
4. 按重要程度分类整理：
   📌 重大事项（停牌、重组、增发、业绩预告/快报）
   📊 研报评级（买入/增持/下调，含目标价）
   📋 常规公告（董事会决议、股东大会、股份变动等）
5. 每条标注：股票名称、公告标题、来源、时间、摘要（不超过 100 字）
6. 如果某只自选股今日无公告也无研报，不列出
7. 整份以 Markdown 格式保存为 reports/公告研报-YYYY-MM-DD.md
8. 把该 .md 文件通过微信发给我
9. 注意：红色=涨、绿色=跌的中国惯例
```

### 时间设置
```
0 18 * * 1-5
时区：Asia/Shanghai
```

### 推送方式
```
微信推送 .md 文件
```

## 任务 3：学习助手（长期指令，随时响应）

- **类型**：长期行为指令（不是定时任务）
- **配置方式**：在微信给龙虾发一条消息，龙虾会记住并长期生效

### 微信发送内容（直接复制粘贴发给龙虾）

```
请记住以下长期指令：

从现在起，当我的消息以「学习：」开头时，你切换为学习助手模式，执行以下流程：
1. 理解我要学习的知识点（如「学习：二次函数的图像性质」）
2. 用 Tavily web_search 搜索该知识点的教学资料，优先选择教育类网站
3. 整理成结构化学习笔记，包含：
   📖 知识点概述（1-2 句话解释是什么）
   📝 核心概念（分点列出关键定义和公式）
   💡 典型例题（2-3 道带详细解题步骤）
   ⚠️ 易错点（常见误区提醒）
   🔗 延伸阅读（相关知识点链接）
4. 以 Markdown 格式保存为 reports/学习笔记-知识点-YYYY-MM-DD.md
5. 把该 .md 文件发我微信

当消息不以「学习：」开头时，正常按金融助手模式响应。
```

### 使用方式
```
在微信发：学习：二次函数的图像性质
在微信发：学习：艾略特波浪理论入门
在微信发：学习：宏观经济学的GDP概念
```

## 任务 4：日常问答机器人（长期指令，随时响应）

- **类型**：长期行为指令（不是定时任务）
- **配置方式**：在微信给龙虾发一条消息

### 微信发送内容（直接复制粘贴发给龙虾）

```
请记住以下长期指令：

从现在起，当我的消息以「问：」开头时，你切换为日常问答模式，快速回答我的问题：
1. 识别问题类型（天气/翻译/汇率/计算/百科/新闻等）
2. 根据问题类型选择最快的数据获取方式：
   - 天气/汇率/航班等实时信息：用 Tavily web_search 搜索
   - 翻译：直接翻译，不需搜索
   - 计算：直接计算，不需搜索
   - 百科知识：用 Tavily web_search 搜索后总结
   - 股票行情：用 westock-data 或 Infoway API 获取
3. 回答要求：简洁、直接、不超过 200 字，先给答案再给补充说明
4. 不需要保存文件，直接在微信回复即可

当消息不以「问：」开头时，正常按之前的模式响应。
```

### 使用方式
```
在微信发：问：今天上海天气怎么样
在微信发：问：100美元等于多少人民币
在微信发：问：帮我翻译"market capitalization"
在微信发：问：2的10次方是多少
```

## 部署顺序

1. 先在面板配「盘中异动监控」和「公告研报速递」两个 cron 任务
2. 然后在微信分别发送任务 3 和任务 4 的长期指令
3. 发一条测试消息验证：
   - 发「问：测试」看日常问答是否生效
   - 发「学习：测试」看学习助手是否生效
4. 周一开盘后观察盘中异动监控是否正常触发
